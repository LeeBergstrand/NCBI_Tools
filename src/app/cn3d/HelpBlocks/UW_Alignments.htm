Import Window : Alignments Commands

  • Merge single
  • Merge to neighbor
  • Merge all
  • Delete single
  • Delete all

Merge single

Select this item and then click on an import sequence to merge it into the end of the multiple alignment. Note that the merge will succeed if and only if all blocks of the multiple alignment are contained in blocks of the import alignment, without insertions. An unmergeable import will be marked with a red background to indicate where the alignment is incompatible.

Merge to neighbor

Similar to merge single except that the sequence is positioned in the multiple alignment right after its closest neighbor (by simple sum of BLOSUM62 scores).

Merge all

Similar to merge single, except all imports that are compatible with the multiple alignment are merged in.

Delete single

Select this item and then click on an import to delete it.

Delete all

Removes all import alignments from the window.
