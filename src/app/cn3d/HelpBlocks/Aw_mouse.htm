Sequence Window : Mouse Mode Commands

  • Select rectangle
  • Select columns
  • Select rows
  • Select blocks
  • Horizontal drag
  • Move row

Select rectangle

Use this mode to select residues in an arbitrary rectangle.

Select columns

Use this mode to select whole columns of the alignment.

Select rows

Use this mode to select whole rows of the alignment.

Select blocks

Use this mode to select entire blocks from the alignment.

Horizontal drag

When the editor is on, use this mode to drag residues and block boundaries horizontally in order to change the alignment.

Move row

When the editor is on, use this mode to drag whole rows up and down, to change the order of sequences in the alignment.
