Usage: sra-dbcc [options] table

    -5|--md5        Check components md5s if present, fail otherwise,
                    unless other checks are requested (default: yes)
    -b|--blob-crc   Check blobs CRC32 (default: no)
    -x|--exhaustive Continue checking table for all possible errors
                    ( default is to exit after first error )

    -h|--help       This help
    -V|--version    Program version
