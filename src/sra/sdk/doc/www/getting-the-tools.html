The SRA toolkit is available in the form of pre-built executable binaries or in source code format.

Windows

NCBI does not support the download and building from source for the Windows operating system as Windows is not usually configured for building programs and when it is there is little standardization of what external libraries are available.

Linux and Mac

NCBI is fully tested on both of these platforms.

Other platforms

NCBI does not officially support other platforms and con not offer support.
